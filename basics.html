Basics of HTML

1. Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It uses a system of tags to structure content such as text, images, links, videos, and more on the web.

Every HTML document follows a basic structure. Understanding this structure is the foundation of every website.

2. <!DOCTYPE html>

  • This is called the document type declaration.
  • It tells the browser that the file is written in HTML5, which is the latest standard.
  • It must appear at the very top of the HTML file before the <html> tag.
<!DOCTYPE html>

Think of it as the document's "ID card" to tell the browser how to read the code.

3. <html> Tag

  • This is the root element of any HTML document.
  • It wraps all other HTML elements except <!DOCTYPE>.
<html>
  <!-- All HTML content goes here -->
</html>

It marks the beginning and end of the HTML code.

4. <head> Tag

  • Contains metadata and settings for the web page.
  • Not visible on the main page, but critical for browsers and SEO.
  • Includes elements like <title>, <meta>, <link>, and <style>.
<head>
  <title>My First Page</title>
  <meta charset="UTF-8">
</head>

5. <title> Tag

  • Defines the title of the web page shown in the browser tab.
  • Important for SEO and user experience.

7. <body> Tag

  • Contains everything visible to the user: text, images, buttons, etc.
  • Goes directly after the </head> tag.
  • Only one <body> tag per HTML page.
